# Four fours calculator

```
0 = 44-44
1 = 44/44
2 = 4-(4+4)/4
3 = (4+4+4)/4
4 = 4+4*(4-4)
5 = (4+4*4)/4
6 = 4+(4+4)/4
7 = 44/4-4
8 = 4+4+4-4
9 = 4+4+4/4
10 = (44-4)/4
```

## How to play

```sh
go run ./cmd/four-fours      # This prints ordinary four-fours
go run ./cmd/four-fours 9999 # This prints four-nines
```
